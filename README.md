## CSmarch
本项目用于结构化记录对于计算机相关学科的学习内容，主要组成为书籍和课程。

整体内容的参照ACM和IEEE计算机学会的[计算机科学课程体系规范（2013）](https://www.acm.org/education/curricula-recommendations)进行组织。考虑到知识体系表达的完整性，未作深入学习的领域也在标题中写出。

Keep reading, keep learning.
